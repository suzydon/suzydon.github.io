Markets & Trading Strategy
Systematic FX Momentum Strategy & Backtesting Framework
Suzy Tang
Quantitative Research, 2026
code

Key finding: FX momentum delivered no risk-adjusted return post-2010 (out-of-sample Sharpe -0.3). Systematic time-series momentum strategy across seven G10 currency pairs, built as a rigorous research backtest on real market data (2010–present). Implements volatility-targeted position sizing, explicit transaction costs, strict no-look-ahead signal lagging, and in-sample / out-of-sample validation with a lookback robustness sweep. The framework finds FX momentum has largely failed to deliver risk-adjusted returns post-2010—consistent with the academic literature—demonstrating disciplined backtesting and honest evaluation over curve-fitted performance.
